# SoSly UI Tweaks

A FoundryVTT module for SoSly UI customizations.

## Installation

Install via the FoundryVTT module browser or use the manifest URL:
```
https://github.com/sosly/foundryvtt-sosly-ui-tweaks/releases/latest/download/module.json
```

## Compatibility

- FoundryVTT v12 - v14
